5.1 JavaScriptにおけるオブジェクト指向の特徴

■クラシックなやり方
「クラス」は用意されていないので、「プロトタイプ」を使う。
▼宣言の仕方がわかるコード
var Member = function(firstName, lastName) {
this.firstName = firstName;
this.lastName = lastName;
this.getName = function() {
return this.lastName + ' ' + this.firstName;
}
};

var mem = new Member('祥寛', '山田');
console.log(mem.getName());

thisキーワードはこの場合「生成したインスタンスの」という意味。
※thisが何を指すのか場面ごとに表した表が本の中にあるよ（229ページ）

5.3 スーパークラス・サブクラス - プロトタイプチェーン -

5.3.1 プロトタイプチェーンの基礎

▼動作がわかるコード
var Animal = function() {};

Animal.prototype = { //スーパークラスの宣言
walk : function() {
console.log('トコトコ...');
}
};

var Dog = function() { //これの意味はテキストで確認（246ページ）
Animal.call(this);
};
Dog.prototype = new Animal(); //ここでスーパークラスを継承！
Dog.prototype.bark = function() {
console.log('ワンワン！ ');
}

var d = new Dog();
d.walk();
d.bark();

※tokotkoは動物みんなが行う。barkは犬だけが行う。
