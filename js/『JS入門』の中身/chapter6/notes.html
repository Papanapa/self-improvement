ノードウォーキング

一般的な使い分け
getなんちゃら/queryなんちゃらで特定の要素を取得した後、近接するノードはノードウォーキングで取得する

基本：child_nodes

▼動作がわかるコード（コピーして.htmlで保存してブラウザで実行し、コンソールを確認してみて）
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8" />
<title>JavaScript本格入門</title>
</head>
<body>
<form>
  <label for="food">一番好きな食べ物は？：</label>
  <select id="food">
    <option value="ラーメン">ラーメン</option>
    <option value="餃子">餃子</option>
    <option value="焼き肉">焼き肉</option>
  </select>
<input type="submit" value="送信" />
</form>
<script>
var s = document.getElementById('food');
var opts = s.childNodes;
for (var i = 0, len = opts.length; i < len; i++) {
  var opt = opts.item(i);
  if (opt.nodeType === 1) {
    console.log(opt.value);
  }
}

</script>
</body>
</html>

firstElementChild（子を取得）とnextElementSibling（兄弟を取得）を使った方法はテキストを見て。
