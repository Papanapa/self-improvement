技術的メモ

else ifの使い方

（何故かelse ifはJSで使えないかと思ってた...）
▼コード
var x = 30;
if(x >= 20){
console.log('変数xは20以上です。');
} else if (x >= 10) {
console.log('変数xは10以上です。');
} else {
console.log('変数xは10未満です。');
}

for...in命令の使い方

連想配列（オブジェクト）に一個一個何かの処理をするとき便利。 ▼コード
var data = { apple:150, orange:100, banana: 120 };
for (var key in data) {
console.log(key + '=' + data[key]);
}

※ただの配列の列挙は次のfor...ofまたはfor文がおすすめ

for...of命令の使い方

▼コード
var data = [ 'apple', 'orange', 'banana' ];
Array.prototype.hoge = function () {}
for (var value of data) {
console.log(value);
}

※console.log(data[value])になっていないところがみそ
